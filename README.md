# DocAppoint
---
*A platform to find the doctor you needed using Angular and Bootstrap*

### Requirements
- Node.js STB
- Git
- Typescript

### Development
``npm i``
``npm start``

### Production
``npm i``
``npm run build``
``npm start --prod``

### Dependencies
- Angular 12.2.7
- Bootstrap 5.1.3
- Animate.css 4.1.1
